# Koru Mat Website

The official website for Koru Mat - a patented fold & roll yoga mat that keeps your practice side separate from the floor side.

## Features

- Responsive design for all devices
- Clean, minimalist interface
- Color options display for yoga mats
- Direct Amazon integration for purchases
- Contact form for customer inquiries
- Instagram feed integration

## Customizations Made

- Removed promotional banners and Shopify branding
- Simplified contact information
- Custom color picker for product variants
- Streamlined navigation
- Removed unnecessary e-commerce elements

## Local Development

To view the site locally:

```bash
# Using Python 3
python3 -m http.server 8000

# Then visit http://localhost:8000 in your browser
```
